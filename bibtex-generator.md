## DOI Metadata Tool

Generate clean BibTeX directly from one DOI or many.

Paste DOI values, DOI URLs, or raw identifiers. The app normalizes each DOI, fetches publication metadata from scholarly registries, and builds BibTeX entries that keep the DOI field and exclude abstract content.

This tool works directly with [Crossref](https://www.crossref.org/) metadata APIs.

It does not generate references by itself and does not invent citation data. It only formats BibTeX from metadata returned by trusted registries.
